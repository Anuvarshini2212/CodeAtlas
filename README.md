# 🤖 CodeAtlas

> **AI-Powered Repository Intelligence Platform**

CodeAtlas is a full-stack AI-powered platform that helps developers understand software repositories by generating intelligent project summaries, AI-powered file explanations, repository insights, and interactive codebase conversations.

---

## 🚀 Features

* 🔐 Secure User Authentication (JWT)
* 📂 Upload GitHub Repository ZIP Files
* 🤖 AI-Generated Project Overview
* 📄 AI File Summaries
* 💬 Repository Chat using AI
* 🌳 Interactive File Explorer
* 📊 Repository Metrics Dashboard
* 🗑 Delete & Manage Repositories
* 🎨 Modern Responsive UI

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/Anuvarshini2212/CodeAtlas.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd server
npm install
npm run dev
```

---

## 🔑 Environment Variables

Create a `.env` file inside the `server` folder.

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
GROQ_API_KEY=your_api_key
```
